# Student Attendance System

Backend prototype for a school/college attendance management system.
Designed to run on a teacher/admin system with controlled access.

## Current Features (Phase 1)
- Working day calculation
- Automatic exclusion of Saturdays and Sundays
- Core attendance logic (foundation)

## Tech Stack
- Python 3
- File-based storage (JSON)

## Planned Features
- Indian national holidays exclusion
- Subject-wise attendance
- Student & teacher roles
- Login & authentication
- Local server (FastAPI)
- PostgreSQL database
- Cloud deployment (SaaS model)
